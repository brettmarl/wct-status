# 2026 TRAIL RESERVATIONS
Each year, Parks Canada opens reservations on a specific date in January, known as reservation day. This is when thousands of hikers all try to book permits at the same time. The most popular dates often sell out within minutes. For 2026, Reservation Day has been changed to Thursday February 5th at 8am PST.
